# apps/web

Next.js user-facing app untuk Bisakerja.

## Commands

- `pnpm dev`
- `pnpm lint`
- `pnpm typecheck`
- `pnpm test`
- `pnpm build`

## Structure

- `src/app` untuk routes (App Router)
- `src/features` untuk modul domain
- `src/components` untuk komponen reusable
- `src/services` untuk API/service clients
- `src/lib` untuk util dan type umum
- `tests` untuk baseline unit/component tests
