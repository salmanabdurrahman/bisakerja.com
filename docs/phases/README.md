# Phases & Iterations Hub

Dokumen phase memecah implementasi menjadi milestone yang terukur, dengan gate kualitas yang enforceable.

- [Implementation Kickoff Guide](./implementation-kickoff.md)
- [Implementation Roadmap](./implementation-roadmap.md)
- [Implementation Checklist](./implementation-checklist.md)

## Aturan Operasional

1. Setiap phase hanya boleh ditutup jika acceptance criteria phase terpenuhi.
2. Semua gate wajib harus pass sesuai `docs/standards/ci-quality-gates.md`.
3. Status `✅ Implemented` pada checklist wajib memiliki evidence lengkap (code, test, CI, docs/ADR).
4. Mulai implementasi bertahap wajib mengikuti urutan dan prasyarat di kickoff guide.
