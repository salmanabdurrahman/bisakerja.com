## Aturan Operasional

1. Setiap phase frontend harus memenuhi gate required sebelum ditutup.
2. Status `✅` pada checklist frontend wajib punya evidence lengkap (code, test, CI, docs/ADR).
3. Roadmap frontend harus tetap sinkron dengan gate dan acceptance criteria pada standar engineering.
4. Kickoff frontend mengikuti urutan implementasi bersama backend pada kickoff guide.
